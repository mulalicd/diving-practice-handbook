# Abbreviations

This document lists all abbreviations used in the Diving Practice Handbook and Web App.

| Abbreviation | Meaning |
|--------------|---------|
| SCUBA        | Self-Contained Underwater Breathing Apparatus |
| NDL          | No-Decompression Limit |
| O2           | Oxygen |
| N2           | Nitrogen |
| PPO2         | Partial Pressure of Oxygen |
| DCS          | Decompression Sickness |
| RDP          | Recreational Dive Planner |
| ACA          | AI Coding Assistant |
| GDPR         | General Data Protection Regulation |
| PSI          | Pounds per Square Inch (pressure) |
| ATM          | Atmosphere (pressure unit) |
| FRC          | Freediving Recovery Check |
| PADI         | Professional Association of Diving Instructors |
| SSI          | Scuba Schools International |
| CMAS         | Confédération Mondiale Des Activités Subaquatiques |
