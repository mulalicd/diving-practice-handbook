# Interactive Exercises

## Purpose
Hands-on practice materials for developing diving skills through structured scenarios and calculations.

## Exercise Categories

### 1. Philosophy & Self-Reflection
- File: `philosophy-exercises.md`
- Focus: Developing self-awareness and reflective practice

### 2. Dive Table Practice
- File: `dive-table-practice.md`  
- Focus: Manual dive planning and decompression calculations

### 3. Risk Assessment Scenarios
- File: `risk-assessment-scenarios.md`
- Focus: Real-world decision-making and Go/No-Go judgments

### 4. Equipment Configuration Labs
- File: `equipment-configuration-labs.md`
- Focus: Redundancy planning and configuration logic

### 5. Gas Calculations
- File: `gas-calculations.md`
- Focus: PPO2, nitrogen loading, and gas planning

## How to Use Exercises
1. Read related handbook chapter first
2. Attempt exercises independently
3. Check solutions and reasoning
4. Apply learnings to real dive planning
5. Discuss with dive buddies or instructors

## Safety Note
All exercises are educational simulations. Never apply untested plans to real dives without professional verification.
